# NewsApp
Consumes newsapi.org api

## Features

- Material Design Guidelines
- Constraint Layout
- Retrofit
- RecyclerView
- MVVM Application Architecture(LiveData + ViewModel +DataBinding)
- Navigation
- Room Database and Repository Pattern
- Testing
- Dependency Injection with Koin
